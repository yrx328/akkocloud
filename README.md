## AkkoCloud 是什么来头？

2019年成立的国人主机商，一开始主要做国内独立服务器，后来扩展到海外 VPS。目前主打三个机房：**美国圣何塞、德国法兰克福、英国伦敦**，全部采用 KVM 架构 + SSD 硬盘。

最大的卖点就一个：**全线 CN2 GIA 线路**。电信联通去程 GIA，三网回程也是 GIA，移动用户的话德国和英国机房走 CMIN2 高端线路。换句话说，晚高峰骨干网爆炸的时候，他家机器理论上受到的影响会小很多。

界面全中文，支付宝付款，对国内用户比较友好。客服工作时间是中午12点到晚上10点，不是24小时，这点需要留意。

## 三大机房线路情况

**美西圣何塞**：三网回程 CN2 GIA，电信联通去程走 CN2 GIA。延迟约 168ms 左右，对大多数国内用户来说是最优选。美西机房还可以分配原生 IP 或提供免费 DNS 解锁，Netflix、Disney+、YouTube Premium 等主流流媒体都能搞定。

**德国法兰克福**：三网回程 GIA，电信联通去程 GIA，移动走 CMIN2 直连。延迟约 180ms，相比普通欧洲 VPS 快了不少，丢包率也低。

**英国伦敦**：三网回程 GIA，电信联通去程 GIA，移动 CMIN2 双向。延迟约 200ms，是三个机房里最远的，但胜在线路质量稳。

## 几点需要注意的

**流量是双向计算的**，上传下载都算在流量里，实际使用的时候要注意。512M 内存那档主要适合轻量应用，跑重型服务可以往 768M 或 1G 的套餐走。

**不支持退款**，这点比较硬。建议先季付试水，满意了再年付。

**客服不是 24 小时**，工作时间中午到晚上，如果半夜出问题，要等到第二天处理。

**目前只支持支付宝**，没有信用卡或者其他支付方式。
